# IA Infra Latino Static Site

Sitio estático temporal para GitHub Pages mientras se desarrolla la página oficial de IA Infra Latino.

## Stack

- HTML
- CSS
- JavaScript
- GitHub Pages

## Paleta

- Azul eléctrico: #2563EB
- Negro grafito: #111827
- Cyan IA: #06B6D4
- Gris técnico: #9CA3AF

## Deploy en GitHub Pages

1. Crear repositorio: `ia-infra-latino-site`
2. Subir todos los archivos de esta carpeta.
3. Ir a Settings → Pages.
4. Source: Deploy from branch.
5. Branch: `main` / root.
6. Guardar.

## Dominio personalizado

Cuando estés listo, configura:

```txt
iainfralatino.com
```

en GitHub Pages y luego ajusta DNS en Namecheap.
